Linear System

A System has input and output (Function, Transformation, Operator).

Linear System: $F(k1 * x1 + k2 * x2) = k1 * F(x1) + k2\ * F(x2)$

  • Preserving Multiplication
    • F(kx) = ky (Assuming F(x) = y)
  • Preserving Addition
    • F(x1 + x2) = y1 + y2 (Assuming F(x1) = y1, F(x2) = y2)
